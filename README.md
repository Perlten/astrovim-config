# astrovim-config
## Setup

``` console
mv ~/.config/nvim ~/.config/nvimbackup && git clone https://github.com/AstroNvim/AstroNvim ~/.config/nvim && git clone https://github.com/Perlten/astrovim-config.git ~/.config/nvim/lua/user && nvim  --headless -c 'autocmd User PackerComplete quitall' -c 'PackerSync'
```
